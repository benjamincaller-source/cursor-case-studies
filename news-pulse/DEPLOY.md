## Activer GitHub Pages (une seule fois)

Si le lien renvoie une 404, activez Pages dans GitHub :

1. Ouvrez https://github.com/benjamincaller-source/cursor-case-studies/settings/pages
2. **Source** → Deploy from a branch
3. **Branch** → `gh-pages` / `/ (root)`
4. Cliquez **Save**

Le déploiement se met à jour automatiquement à chaque push sur `main`.
